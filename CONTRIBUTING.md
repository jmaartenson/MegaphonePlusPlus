## How to contribute

This repository is a maintained public fork of Tim Neill's original Megaphone++ source:

- Maintained fork: https://github.com/jmaartenson/MegaphonePlusPlus
- Original source: https://github.com/timneill/megaphoneplusplus

Contributions to this fork are accepted under the same GPL-3.0 license as the original project.

#### **Did you find a bug?**

* **Ensure the bug was not already reported** by searching on GitHub under [Issues](https://github.com/jmaartenson/MegaphonePlusPlus/issues).

* If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/jmaartenson/MegaphonePlusPlus/issues/new). Be sure to include a **title and clear description**, as much relevant information as possible, including what happened and what you were doing when it happened. A screenshot also helps.

#### **Did you write a patch that fixes a bug?**

* Open a new GitHub pull request with the patch.

* Ensure the PR description clearly describes the problem and solution. Include the relevant issue number if applicable.

#### **Did you fix whitespace, format code, or make a purely cosmetic patch?**

Changes that are cosmetic in nature and do not add anything substantial to the stability or functionality will not be accepted.
